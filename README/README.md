# Документація проекту TRM

Ця папка містить всю документацію проекту Tiny Recursive Model (TRM).

## Структура документації

### Основні документи

- **[AUDIT_REPORT.md](AUDIT_REPORT.md)** - Детальний аудит проекту, виявлені проблеми та рішення
- **[REFACTORING_SUMMARY.md](REFACTORING_SUMMARY.md)** - Підсумок рефакторингу проекту
- **[SCRIPTS_AUDIT_REPORT.md](SCRIPTS_AUDIT_REPORT.md)** - Аудит всіх скриптів проекту

### Керівництва

- **[CHECKPOINT_GUIDE.md](CHECKPOINT_GUIDE.md)** - Керівництво з роботи з checkpoint'ами
- **[PARAMETERS_CHECK.md](PARAMETERS_CHECK.md)** - Перевірка параметрів навчання

## Швидкий доступ

### Для початку роботи
1. Прочитайте головний [README.md](../README.md) в корені проекту
2. Перегляньте [CHECKPOINT_GUIDE.md](CHECKPOINT_GUIDE.md) для роботи з checkpoint'ами

### Для розробників
1. [AUDIT_REPORT.md](AUDIT_REPORT.md) - повний огляд структури проекту
2. [REFACTORING_SUMMARY.md](REFACTORING_SUMMARY.md) - історія змін та покращень
3. [SCRIPTS_AUDIT_REPORT.md](SCRIPTS_AUDIT_REPORT.md) - опис всіх скриптів

### Для налаштування
1. [PARAMETERS_CHECK.md](PARAMETERS_CHECK.md) - перевірка параметрів навчання

## Інші файли документації

- `../logs/README.md` - Документація по логах
- `../datasets/README.md` - Документація по датасетам
