# Навчання з інтерактивним тестуванням

## Опис

Скрипт `start_training_with_test.sh` запускає навчання моделі та автоматично пропонує протестувати модель після кожного checkpoint.

## Використання

```bash
# Зробити скрипт виконуваним
chmod +x scripts/start_training_with_test.sh

# Запустити навчання з тестуванням
./scripts/start_training_with_test.sh
```

## Як це працює

1. **Запуск навчання** - скрипт запускає навчання через `runtime.bootstrap`
2. **Моніторинг checkpoint'ів** - скрипт моніторить папку `checkpoints/` на нові файли
3. **Пропозиція тестування** - після кожного нового checkpoint:
   - Питає "Протестувати модель? (y/n/f/q)"
   - Timeout: 5 секунд (якщо немає відповіді - продовжує навчання)
4. **Тестування**:
   - `y` - швидкий тест (10 прикладів) з прогрес-баром
   - `f` - повний тест (всі приклади) з прогрес-баром
   - `n`/`q` - пропустити тестування

## Типи тестування

### Швидкий тест (y)
- Тестує на 10 прикладах
- Показує прогрес-бар у відсотках
- Швидко (1-2 хвилини)
- Метрики: точність, середня схожість

### Повний тест (f)
- Тестує на всіх прикладах з eval датасету
- Показує прогрес-бар у відсотках
- Довше (5-10 хвилин)
- Детальні метрики

## Налаштування

У скрипті можна змінити:
- `TEST_INTERVAL=5` - час очікування відповіді (секунди)
- `QUICK_TEST_SAMPLES=10` - кількість прикладів для швидкого тесту

## Зупинка

Натисніть `Ctrl+C` для зупинки навчання та тестування.
